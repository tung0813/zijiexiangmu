# 项目使用说明

## 环境要求

- Node.js >= 18.0.0
- npm >= 9.0.0

## 本地开发

### 1. 安装依赖

```bash
npm install
```

### 2. 配置环境变量

创建 `.env.local` 文件（参考 `.env.example`）：

```env
VOLCENGINE_API_KEY=你的火山引擎API密钥
VOLCENGINE_ENDPOINT=https://ark.cn-beijing.volces.com/api/v3
VOLCENGINE_MODEL=你的模型端点ID
```

### 3. 启动开发服务器

```bash
npm run dev
```

访问 http://localhost:3000

## 功能使用

### 创建新对话

1. 点击左侧"新建对话"按钮
2. 系统自动创建新的对话会话

### 生成商品素材

1. **上传商品图片**（可选）
   - 点击上传区域
   - 选择商品主图（支持多张）
   - 最多可上传 3 张图片

2. **输入商品描述**
   ```
   雪莲果，来自云南高原，脆爽清甜，水润多汁，经过超200项检测，坏果包赔
   ```

3. **点击发送**
   - AI 将自动生成：
     - 商品标题（10-30字）
     - 商品卖点（3-5个）
     - 主图氛围文案（8-15字）

### 迭代优化素材

在同一对话中继续输入优化建议：

```
标题太长了，简化到20字，突出"云南"和"清甜"
```

系统会基于之前的内容进行优化。

### 复制素材

点击每个素材卡片右上角的"复制"按钮，内容将自动复制到剪贴板。

### 添加水印（开发中）

未来版本将支持为图片添加"抖音电商前端训练营"水印。

## 常见问题

### Q: 生成失败怎么办？

A: 检查以下几点：
- 火山引擎 API 密钥是否正确
- 网络连接是否正常
- API 余额是否充足

### Q: 如何切换不同的 AI 模型？

A: 修改 `.env.local` 中的 `VOLCENGINE_MODEL` 为其他模型端点 ID。

### Q: 对话历史保存在哪里？

A: 对话历史保存在本地 SQLite 数据库中（`data/database.sqlite`）。

### Q: 如何清空所有对话？

A: 删除 `data/database.sqlite` 文件，重启服务即可。
